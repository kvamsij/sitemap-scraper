Project: "Sitemap Scraper"

Description:

    Develop a sitemap scraper that accepts a domain name as input, reads the `robots.txt` file, and extracts sitemap URLs. The scraper should handle both standard and `.gz` compressed sitemaps, filter product-specific sitemaps, and log events while writing the extracted data to a CSV file. The scraper should also filter URLs based on disallowed paths in `robots.txt` and identify product URLs using configurable patterns.

Setup:

1. Install the following dependencies:
   - `typescript` for type safety.
   - `jest` for unit testing.
   - `axios` for HTTP requests.
   - `ts-node` for running TypeScript files directly.

2. Ensure the project is initialized with a `package.json` file and includes scripts for building and running the application.

Development Plan:

1. **Input Handling**:
   - Create a class to accept and validate domain name input from the terminal.

2. **Fetching `robots.txt`**:
   - Create a class to fetch the `robots.txt` file from the provided domain.

3. **Parsing `robots.txt`**:
   - Create a class to parse the `robots.txt` file to:
     - Extract allowed and disallowed paths.
     - Identify and extract sitemap URLs.

4. **Filtering Product Sitemaps**:
   - Create a class to filter and identify product-specific sitemaps from the extracted sitemap URLs.

5. **Fetching URLs from Sitemaps**:
   - Create a class to fetch all URLs listed in both standard and `.gz` compressed sitemap files.
   - Handle nested sitemaps by recursively fetching and parsing them.

6. **Filtering URLs**:
   - Implement a class to filter URLs based on:
     - Disallowed paths in `robots.txt`.
     - Configurable product URL patterns (e.g., `/product/`, `/item/`, `/sku/`, `/p/`, `/products/`, and query parameters like `?productId=`).

7. **Optimizing URL Extraction**:
   - Implement parallel processing or batching to speed up the extraction of URLs from large sitemaps.

8. **Logging System**:
   - Implement a centralized logging system using `winston` to log events, errors, and application flow.

9. **Writing to CSV**:
   - Create a class to write the extracted URLs to a CSV file.

